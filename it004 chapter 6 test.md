###### Bài 1:
Cho F = {A -> B, BC -> D}, chứng minh AC -> D

> [!info] answer
>+ BC -> D ==> B -> D, C -> D
>+ A -> D, C -> D ==> AC -> D

###### Bài 2:
Cho F = {A -> BC, AC -> D}, chứng minh AC -> BCD
> [!info] answer
> + A -> BC => AC -> BC
> + AC -> D, AC -> BC => AC -> BCD

###### Bài 3:
Cho F = {CD -> H, B -> EG, E -> AD}, chứng minh BC -> H
> [!info] answer
> + B -> EG => B -> E, B -> G
> + B -> E, E -> AD => B -> AD
> + B -> AD => B -> A, B -> D
> + B -> D, CD -> H => BC -> H

###### Bài 4:
Cho F = {AB -> C, B -> D, CD -> E, CE -> GH}, chứng minh AB -> GH
> [!info] answer
> AB -> C => AB -> BC
> B -> D => BC -> CD
> CD -> E => CD -> CE
